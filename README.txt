# 1C  Решаемая проблема, цель, задачи работы
● Проблема — высокие временные затраты на создание документации и отчетности
● Цель работы — создать конфигурацию, уменьшающую временные затраты по
созданию заявок и отчетности
● Задачи работы
– Быстрая регистрация заявок на проведение работ;
– Регистрация результатов проверки;
– Создание отчетов по проведенным работам и итоговым отчетам за требуемый
период.
3
Концепция / теория решения кейса
● Теоретическое обоснование работы — основой концепции послужат «Модель
данных».
● Методы решения кейса — поиск аналогичных работ по внедрению СУБД для
упрощения документооборота.
● Научная новизна — взамен разрозненных информационных систем, предлагается
объедененная информационно-техническая система, которая включает в себя
создание всех необходимых отчетов и регистрации заявок. 
Результаты работы. Предложения для практического
использования
● Результаты работы и итоговое решение проблемы — на платформе 1С Предприятие 8.3
сформированная конфигурация для регистрации заявок и автоматической выдаче
заключений. Поиск, обработка, хранение информации в данной информационной системе
будет удобней чем в рукописных журналах, набранных в отдельные документы
заключений о качестве. Все будет храниться в единой системе, что ускорит процесс
взаимодействия с информацией. Безопасность системы обеспечивается на уровне
подключения к локальной сети и к базе данных 1С. Работа нескольких штатных единиц
будет осуществлена через локальную сеть и удаленный рабочий стол. Печать, сохранение
в форматируемом формате excel предусмотренна на уровне ядра. Для наглядного
представления в прототипе отображена только табличная часть отчета, полная форма
также реализуема. По желанию ответственного за кейс был включен возможность
добавления схем контроля.
● Обоснование эффективности предлагаемого решения — настройка локальной сети и
удаленных рабочих доступов осуществима без затрат, стоимость программ 1С окупается
меньше чем за 11 месяцев. При необходимости данная конфигурация может быть
усовершенствована сторонними 1С программистами под нужды компании.
● Моделирование процесса внедрения предлагаемого решения — после внедрения
конфигурации в систему, время обработки заказов и результатов снизится в полтора раза,
а выдача заключений, создание архивных журналов будет в автоматическом порядке, что
сократит общее время работы почти в 3 раза.
